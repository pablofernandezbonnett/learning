# Labs

This folder is for runnable companion material that supports the notes in
`topics/`.

The preferred shape is:

- small
- easy to run locally
- focused on one concept or flow
- useful as a learning aid, not only as a code dump

Current labs include:

- Java modern language and runtime examples
- Kotlin basics
- Spring Boot sample code
- algorithm practice companions in Java and Kotlin
- Kotlin backend examples for correctness, async boundaries, caching, and JVM behavior
